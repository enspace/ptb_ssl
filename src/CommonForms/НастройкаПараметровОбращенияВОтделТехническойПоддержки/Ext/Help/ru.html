Параметры отправки электронного сообщения

Сервис ® Интернет-поддержка пользователей ® Параметры отправки электронного сообщения

В форме заполняются основные данные, необходимые для отправки сообщения в отдел технической поддержки.

Параметры условно разделены на две группы. К первой относятся данные, значения которых используется для отправки электронных сообщений (адрес электронной почты отдела технической поддержки, данные отправителя). Ко второй относятся данные, необходимые сотрудникам отдела технической поддержки для понимания описываемой ситуации (описание используемой конфигурации, параметры компьютеров, на которых установлено 1С:Предприятие 8 и т.д.).

При формировании обращения в отдел технической поддержки в текст обращения на основе введенных параметров будет добавлен текст, содержащий описание параметров. При необходимости данный текст можно отредактировать. Никакой иной информации в обращении не формируется.

Состав страниц зависит от установленных значений параметров раздела "Основные данные по использованию 1С:Предприятие 8 и конфигурации".
Например, если указан клиент-серверный вариант 1С:Предприятие 8, то будет показана страница для ввода данных по компьютеру, на котором установлен SQL-сервер.

Некоторые параметры являются обязательными к заполнению. Поля для ввода этих параметров в незаполненном состоянии имеют красную подчеркивающую линию.

Введенные значения параметров запоминаются и используются при формировании очередного обращения. Если в процессе эксплуатации некоторые данные изменились (например, был заменен компьютер, изменились телефон, наименование организации, фамилия ответственного лица или другие данные), то следует ввести новые данные.

Некоторые параметры формируются программой и включаются в состав обращения автоматически. Это описание используемой конфигурации (наименование, версия, поставщик), параметры локального компьютера, на котором формируется обращение, и данные о версии 1С:Предприятии 8.

С помощью кнопки "Написать сообщение" командной панели открывается форма нового обращения в отдел технической поддержки.
Если форма настройки параметров обращения была открыта из формы обращения, то кнопка "Написать сообщение" будет недоступна.

С помощью кнопки "Печать" командной панели формируется текстовый документ, содержащий текст, включаемый в обращение в отдел технической поддержки. 

Важно! В текст включаются только те параметры (описание используемой конфигурации, параметры компьютеров, на которых установлено 1С:Предприятие 8 и т.д.), по которым указана информация.
